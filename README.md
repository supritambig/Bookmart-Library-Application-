# Bookmart Library Application

A modern, student-friendly online library portal built with React and Vite. Bookmart allows users to browse books, reserve titles, track borrowing history, and discover curated reading lists.

## Features

- **Fast Access**: Find books, notes, and research material in seconds
- **Easy Membership**: Create your account in one step and start borrowing
- **Book Browsing**: Explore a vast collection of over 10,000 books and library resources
- **Reservation System**: Reserve your favorite titles online
- **Borrowing Tracking**: Track due dates and borrowing history
- **User Authentication**: Secure signup and login functionality
- **Responsive Design**: Built with Bootstrap for a seamless experience across devices

## Tech Stack

- **Frontend**: React 18.3.1
- **Build Tool**: Vite 5.4.1
- **Styling**: Bootstrap 5.3.2
- **Language**: JavaScript (ES6+)

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd bookmart-library-application
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173` (default Vite port).

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the project for production
- `npm run preview` - Preview the production build locally

## Usage

1. Visit the application in your browser
2. Create an account using the signup form
3. Explore the library features and browse available books
4. Reserve books and track your borrowing history

## Project Structure

- `src/App.jsx` - Main application component
- `src/main.jsx` - Application entry point
- `src/components/` - Reusable React components (Header, Body, Footer, etc.)
- `public/index.html` - HTML template
- `vite.config.js` - Vite configuration
